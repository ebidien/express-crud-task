# express-crud-task
## Home - Welcome Page
Method `GET`  
URL: `https://express-crud-task.herokuapp.com`

## Create a Contact
Method `POST`  
URL: `https://express-crud-task.herokuapp.com/contacts`  
### Payload 
```
name: string,
email: string,
country: string
```
## Get all Contacts  
Method `GET`  
URL: `https://express-crud-task.herokuapp.com/contacts`  

## Get a Contact  
Method `GET`  
URL: `https://express-crud-task.herokuapp.com/contacts/:id`

## Delete a Contact  
Method `DELETE`  
URL: `https://express-crud-task.herokuapp.com/contacts/:id`
